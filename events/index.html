E3B Events

E3B Events consist of the following:

Event Order:
Test Day 1: EPFA* and Land Navigation*
Test Day 2: Individual Testing Stations
Test Day 3: Individual Testing Stations
Test Day 4: Individual Testing Stations
Test Day 5: 12-Mile Foot March*, Final Event*, and Award Ceremony

* denotes a key event. Failure to pass a key event results in disqualificiation from E3B contention.
